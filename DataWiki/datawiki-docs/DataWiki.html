DataWiki Home

상명대학교 DataWiki입니다! 여기서 수 십여 가지의 다양한 데이터셋을 만나실 수 있습니다. 학습을 위해 여기에 있는 데이터셋을 이용하거나 실제 머신러닝에서 활용하려는 경우, DataWiki는 최적의 사이트가 될 것입니다.

만약 문서 내용이나 링크에 오류가 있는 경우 smdatawikimanager@smu.co.kr 나 대나무숲 문의게시판 으로 의견을 써서 올려주시기 바랍니다.



Sidebar

왼쪽에 위치한 Sidebar는 Dataset을 List로 보여주며, 만약 찾고자 하는 Dataset이 보이지 않는 경우 검색을 수행햐여 찾으실 수 있습니다.
추가적으로 정렬 기능도 제공합니다! Sidebar에 있는 A-Z Category Tutorial Task 를 누르면 각각 이름순, Category순, Tutorial유무, Task순 으로 정렬을 수행하게 됩니다.

여기서 원하는 자료를 찾아 이용하시길 바랍니다!



Dataset

DataWiki에서는 총 60여가지의 다양한 학습용 데이터셋을 제공합니다. 추가적으로 4개의 Category와 50여 가지의 Task를 가지고 있습니다. 추가적으로 데이터에 대한 정보나 튜툐리얼을 받고 싶은 경우, 연결된 링크를 따라가기만 하면 됩니다. 이용자는 편하게 정보를 찾고, 보고, 이용만 하면 됩니다!

DataWiki는 다음과 같은 8가지의 기본적인 Dataset정보를 명시합니다.

  • Dataset Name Dataset의 이름이 표시됩니다.
  • Data Category Dataset이 해당하는 Category가 표시됩니다.
  • Data Field Dataset이 해당하는 Field가 표시됩니다.
  • Data Instance Dataset의 총 개수가 표시됩니다.
  • Dataset Task 해당 Dataset이 무엇을 처리하는지 표시됩니다.
  • Description 해당 Dataset의 설명을 명시합니다.
  • Data Link Dataset을 제공하는 사이트를 연결합니다.
  • Tutorial Link Dataset에 대한 Tutorial을 제공하는 사이트를 연결합니다.
